 # Airline Pricing & Booking Trends Analysis

##  Project Overview
The objective of this project is to analyze pricing patterns and booking trends in the airline industry.
This interactive dashboard helps in understanding how factors like booking urgency, flight duration, and departure time slots influence ticket prices.

##  Key Insights from Dashboard
* *Booking Urgency:* A significant price hike of 30-40% was observed as the departure date approaches.
* *Class Comparison:* Clear visual representation of the pricing gap between Business and Economy classes.
* *Time Slot Impact:* Identified price variations across different time slots (Morning, Evening, and Late Night flights).
* *Airlines Performance:* Comparative analysis of average fares across major airlines like Vistara, Air India, and Indigo.

##  Tech Stack & Tools
* *Microsoft Excel:* Used for data cleaning and advanced analysis.
* *Interactive Dashboards:* Implemented Slicers and Timelines for a dynamic user experience.
* *Data Visualization:* Utilized Line charts, Bar graphs, and Pie charts to represent complex data insights clearly.


## How to Use
1. Download the Excel file from this repository.
2. Open the file and use the *Slicers* (City filters) on the sides.
3. The dashboard will automatically update to show data for your selected routes.
